HISTORY

DEFCONBOTS started in 2004 at DEFCON 12. The goal of the competition has always been to design, build, and program an autonomous robot to complete a specific task.

GET STARTED

Excited to compete but don't know where to start? More interested in software than building the robot? Use our free Reference Design as a starting point.

COMPETE

  • Write some code
  • Show up at the event
